# AWS CodePipeline for WordPress Deployment

This repository demonstrates how to build, test, and deploy a WordPress application in AWS using AWS CodePipeline and CloudFormation templates.

## Files
- `codepipeline.yaml`: CloudFormation template to create the CodePipeline.
- `wordpress-app.yaml`: CloudFormation template for the WordPress application.
- `create-codepipeline.sh`: Script to create the CodePipeline.
- `update-codepipeline.sh`: Script to update the CodePipeline.
